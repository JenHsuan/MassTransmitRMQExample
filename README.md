# MassTransmitRMQExample
## Introduction
It's the example project to integrate ASP.NET core 3.1 and RabbitMQ by using MassTransmit on Service Fabric.

## Preparations
* Rabbit MQ installations
* Visual Studio 2019

## Technical stacks
* ASP.NET core 3.
* Service Fabric
* MassTransmit
* RabbitMQ
